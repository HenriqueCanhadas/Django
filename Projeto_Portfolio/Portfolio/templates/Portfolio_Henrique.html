Quem Sou Eu?

Analista de Infraestrutura e graduando em Ciências da Computação pela Universidade Anhembi Morumbi, com foco em Desenvolvimento Back-end, Front-end e Automação de Processos.

Tenho experiência no desenvolvimento de soluções personalizadas para automação de cálculos e otimização de processos, integrando bases de dados e otimizando tarefas para melhorar a eficiência operacional. Minha principal expertise está no desenvolvimento back-end com Python e Django, além de uma sólida base em front-end com HTML, CSS e JavaScript.

Minhas habilidades também abrangem manutenção de redes corporativas, configuração de dispositivos de rede e suporte de hardware. No momento, estou aprofundando minhas competências em Data Center e em linguagens de programação como Python, Java e C, com o objetivo de consolidar conhecimentos tanto em infraestrutura quanto em desenvolvimento.

Aberto a novos desafios e aprendizagens, tanto em redes quanto em desenvolvimento, busco constantemente aprimorar minhas habilidades para agregar valor estratégico com soluções técnicas eficientes e inovadoras.

Meus Serviços

Projeto Excel

Desenvolvido em Python com Openpyxl, Pandas e Xlwings, este projeto inclui uma interface CustomTkinter para consolidar dados dispersos em várias abas de uma planilha Excel em uma única aba organizada. A automação facilita o acesso e a interpretação dos dados, tornando o processo mais rápido e eficaz.

Projeto Tabelamento

Com uma automação em Python, criei este projeto para otimizar o tabelamento e a organização de dados em planilhas Excel. Usando Openpyxl e Pandas, a solução transforma uma planilha com dados brutos em uma nova versão organizada e padronizada, ideal para facilitar a análise.

Projeto Automação

Este projeto automatiza um processo repetitivo usando Python com Selenium, WebDriver e email.message. Configurada para realizar login em um site, selecionar dados e baixar arquivos a cada 2 horas, a solução também notifica o cliente por e-mail em caso de erro, simplificando o fluxo de trabalho.
